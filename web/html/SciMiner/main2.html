WELCOME TO SciMiner
 


SciMiner is a dictionary- and rule-based biomedical literature mining and functional enrichment analysis tool developed by the Bioinformatics Program, University of Michgan, Ann Arbor.

You may freely modify any web pages of this distributed package.


--Requirement--
1. Java Runtime Environment (JRE) is required on a client's machine to visualize interaction data and network (MiMI).
2. Pop-up should be allowed on this site.
